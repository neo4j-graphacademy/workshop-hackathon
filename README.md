```json
{

		"neo4j-graphacademy": {
			"type": "http",
			"url": "https://mcp.dev.graphacademy.neo4j.com/mcp"

	"inputs": []
}
```

## Commands

1. Enrol me to `workshop-hackathon`
2. Update the .env file  with my instance credentials for `workshop-hackathon`
3. Move to the next lesson in `workshop-hackathon`
